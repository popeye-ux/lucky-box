# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 專案概述

**【校園大冒險】小學生專屬抽籤系統** — 針對小學生設計的互動式網頁抽籤應用，透過幽默生活化的文字與動畫舒緩學習壓力，並引導積極行動。

## 技術規格（待確認）

本專案目前處於需求定義階段，尚未選定框架。建議技術選型：
- **純前端**（無後端需求）：HTML + CSS + Vanilla JS，或 Vue/React
- **無抽籤次數限制**：可無限次抽籤
- **資料來源**：本地 JSON 檔案（60 手籤詩）

## 資料結構

每手籤詩的 JSON Schema（共 60 筆）：

```json
{
  "id": 1,
  "level": "大吉",
  "category": "心動",
  "poem": "詩文內容",
  "meaning": "運勢解讀（白話文）",
  "action_type": "strong | quest",
  "action_content": "能量指令內容"
}
```

- `level`：`"大吉"` / `"中吉"` / `"小吉"` / `"末吉"`
- `category`：`"學業"` / `"校園"` / `"人際"` / `"心動"`
- `action_type`：`"strong"`（強運秘訣，大吉/中吉用）| `"quest"`（改運任務，小吉/末吉用）

## 核心邏輯

1. **抽籤流程**：點擊按鈕 → 播放 2-3 秒動畫 → 從 1-60 隨機整數 → 顯示結果
2. **心動籤判斷**：`id <= 8` 時啟用粉紅泡泡濾鏡 + 心形特效
3. **能量指令翻面**：結果頁下方「啟動能量」按鈕點擊後才顯示 `action_content`

## 視覺規範

- **配色**：馬卡龍色系（粉紅、粉藍、嫩綠、檸檬黃）
- **字體**：圓潤黑體（華康少女體或類似開源字型）
- **動畫**：
  - 大吉 → 畫面震動 + 紙屑噴發
  - 末吉 → 流汗小怪獸，點擊後變身戰士
  - 心動籤 → 粉紅泡泡濾鏡 + 心形特效
- **等級配色**：金色（大吉）、亮橘（中吉）、嫩綠（小吉）、淡藍（末吉）

## 開發起點建議

1. 先建立 `data/fortunes.json`，填入全部 60 手籤詩資料
2. 實作亂數抽籤邏輯
3. 依等級與 category 套用對應動畫與配色
4. 心動籤（ID 01-08）的特殊濾鏡效果獨立處理
